2 axis Ball Balancing Robot

Stable position holding using a dual PID controller on a moving surface.

Project Overview: Stabilizing a Electromechanical System

This project involved designing and implementing a 2-axis Ball and Plate System to continuously balance a ball at a set point. The core challenge was achieving stable position holding on a dynamic, moving surface despite external disturbances. This system serves as a powerful demonstration of closed-loop control in a complex electromechanical environment.

Implementation Details

To manage the system, a **dual PID controller** was designed and precisely tuned to maintain the ball at the set point. This was essential for effectively stabilizing the 2-axis platform against disturbances.

The core control logic was developed using **Arduino C++** firmware, specifically to manage the two independent stabilization loops: **Pitch and Roll**.

For sensing and feedback, **proximity sensors** were utilized to continuously monitor the ball's position, providing the necessary real-time feedback signal to the microcontroller. The platform's tilt movements (X and Y) were accurately actuated by integrating **high-precision stepper motors** via dedicated drivers.

This comprehensive approach successfully demonstrated the control of a complex electromechanical system, achieving **stable position holding** on the moving surface.

Technology Stack (Components Used)

  • Arduino Microcontroller
  • Stepper Motors
  • Proximity Sensors
  • Power Supply
  • LCD Display
  • Jumper Wires
  • Breadboard
